Add start points to existing index

    

This option allows you to add and index a list of start points (usually a new website, or a part of a new website) to an existing index. This can be useful if you manage a list of websites as start points and you wish to add new start points to the index on a regular basis.

It will index the new start point, append this data to the existing index (without having to re-index the existing start points). After a successful incremental index, it will also automatically save the configuration with your newly added start points (so that on your next full re-index, the new start points will be included).

If you are unable to perform an incremental update, please see "Incremental indexing requirements".
