Start directory (offline mode only)

    

The start directory specifies the local directory for the offline scanning to begin in. All sub-directories under the start directory will also be scanned. In other words, you should point this at the folder in which you have created the files for your website.

Advanced start folder options: Clicking on the button will bring up a window which allows you to add other start folders to include in this index. This allows you to index multiple folders (not already under the main start directory) and include them in the search index.
