Notes for users upgrading from Zoom 4.x

    

Due to the introduction of new CSS classes (and the removal of some old CSS classes), we would recommend that you allow Zoom to create a new copy of "search_template.html" in the output directory (that is, remove or rename your existing search template). This way you can view the new default search template, and adopt the CSS styles used to specify the appearance of the recommended links, and other new attributes.

See the "CSS class listing" for more detailed information on the new CSS classes used.
