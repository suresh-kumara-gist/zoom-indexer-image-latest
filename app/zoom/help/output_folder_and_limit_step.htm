Output folder and limit step

    

Zoom will create a set of index files, which will be required for your search function. Specify a folder here for Zoom to create these files in. This can be a temporary folder of your convenience, or a folder under your local copy of your website.

For users of the Professional Edition (or above), you can also specify a limit for the maximum number of pages to index. More limits can be changed in the Configuration window, under the "Limits" tab.

Start indexing immediately

This is the default option and will attempt to index your site as soon as you click Finish for the wizard.

Do not start indexing

Select this option if you have more configuration changes or settings that you would like to make, and would like to start indexing at your own convenience.
