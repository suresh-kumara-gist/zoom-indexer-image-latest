Spider mode

    

Spider mode indexes a remote copy of your website already uploaded and hosted on a web server. It does this via the use of a ‘spider’, which starts from a given start page, and follow the links it finds on each page. This allows the indexer to thoroughly index a website containing both static content (.htm and .html files which do not change) and dynamically generated content (such as websites with a PHP or ASP driven backend, message boards, etc.). It requires an established Internet connection.
