Files required

    

Note that all files required will be created in the Output Directory at the end of indexing.

Filename

Purpose

Notes

search.html

Search page / search template

This is the file to open in your browser to access the search page.

You can also customize the appearance of the search page by modifying this file (see “How do I customize my search page?” for more information.).

 
If this file does not already exists in the output directory, a default template is created.

search.js

Search script

This file contains the core functionality of the searching process.

settings.js

Settings file

This file defines the settings for the search engine.
